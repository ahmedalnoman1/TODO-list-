# TODO-List

A simple task management application where you can add tasks, delete them, or mark them as completed.

---

## 📁 Project Structure

- `index.html` — Main page for the user interface  
- `style.css` — Styles and design of the page  
- `script.js` — JavaScript logic for handling tasks  
- (You can add additional files or folders as needed)

---

## 🚀 How to Run Locally

1. Make sure you have a modern web browser (Chrome, Firefox, Edge, etc.).  
2. Open the `index.html` file in your browser (double-click or use “Open File”).  
3. Start adding and managing your tasks.

---

## 🛠 Features

- Add new tasks  
- Delete tasks  
- Mark tasks as completed  
- Simple and user-friendly interface  

---

## ✨ Future Improvements

- Save tasks in **Local Storage** so they persist after closing the browser  
- Add filters (all tasks, completed, pending)  
- Edit tasks after creating them  
- Connect to a backend/API to store tasks on a server  
- Responsive design for mobile devices
